\subsection{Abbreviations}
\label{sec:Classification_Abbreviations}

\textbf{CART}: Classification and Regression Trees\\
\textbf{MARS}: Multivariate Adaptive Regression Splines\marginnote{MARS is an adaptive procedure for regression, and is well suited for high- dimensional problems.}\\
\textbf{MART}: Multiple Additive Regression Trees\marginnote{MART is the name for regression implementation of Gradient Tree Boosting Algorithm.}\\
\textbf{PRIM}: Patient Rule Induction Method\marginnote{PRIM is also known as Bump Hunting.}\\
